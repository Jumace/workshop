This experiment uses the Flags SDK shape without making Vercel credentials mandatory for local development.

If the project is linked to Vercel and local environment values are pulled, the same flag can be managed through the Vercel dashboard. Without that setup, the demo falls back to `LAB_FLAGS_FALLBACK` or `cyan`.
